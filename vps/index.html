Winserver VPSの特徴Characteristic

最新OSを選択可能

管理者権限を自由に設定するなど、Windows Server 2019で利用可能な最新の機能と技術を活用できます。お客様の環境に応じてOSを選択することが可能です。但し、Windows Server 2016はリソースの消費が大きい為、Windows Server 2019 / 2012 R2 / 2012 を推奨しています。

Hyper-Vを使用

Microsoft社が提供する高機能ハイパーバイザー型仮想化技術であるHyper-Vを使用しています。Hyper-Vを使用すれば、新しい環境構築を準備する際に、必要最小限の時間と手間で実現することが可能です。また、Hyper-V上の仮想マシンのOSは、使用シーンや用途に合わせて最適なOSをインストールできます。

充実の無料サポート

営業時間内は電話での問い合わせを受け付けています。また、メールでのお問い合わせは24時間受け付けており、回答は早ければ当日中、遅くても1営業日以内に回答いたします。専門のスタッフが、お申込み前の相談からサービス開始後の利用方法までサポートします。

メモリ完全保証

ホストサーバーごとに顧客数の制限＆調整を行い、効果的に管理することにより、お客様のメモリを完全保証します。全てのお客様がメモリを最大に使用した場合においても十分な空き容量を確保していますので、常に快適に利用することが可能です。

SSD搭載

従来のHDD（ハードディスクドライブ）より高速処理が可能となったSSD（ソリッドステートドライブ）を搭載しています。データの読み取り、および、書き込み速度を最大100倍高速に実行し、Webサイトのアクセス速度を改善します。

スケールアップ

必要に応じてプランのスケールアップが可能です。「軽めのプランで始めてみたものの、利用しているうちに容量が足りなくなってしまった」といった場合でも安心。迅速にプランのスケールアップに対応します。

稼働率99,99％

サーバー稼働率は99.99％。大阪府内のデータセンターに設置し、24時間監視体制です。大型ディーゼル発電で電力を供給し続けることが可能ですので、非常時の停電でもサーバーが止まることはなく、ビジネス利用にオススメです。

データ保護

RAID50を採用しています。分散パリティ＋ストライピングを介してディスク障害から保護し、緊急時にはデータを安全にバックアップいたします。※お客様環境のバックアップはご自身で行う必要があります。

セキュリティ対策

随時外部からのアタックの遮断を行っており、万全のセキュリティ管理体制です。Windowsファイアウォールにて、お客様の環境に合わせて設定を行うことが可能です。全てのサーバーで小規模なボリュームのDDoS攻撃からの保護します。

Qどのプランを選んだらよいか分からないのですが…

  • A
  • どのプランにするべきかの判断基準は、あなたが必要としているメモリとディスクの容量です。環境に合わせて適切な容量のプランを選ぶことで、無駄なコストを省くことが可能です。どのプランがよいか迷われている方やご不明な点がございましたらお気軽にお問い合わせください。

Qプラン変更は可能ですか？

  • A
  • プランアップにつきましては、現在の環境を引き継いだ状態で行う事が可能です。プランダウンはお受けできませんのでご注意ください。

Q申込みからどれくらいで利用可能ですか？

  • A
  • お申込み完了から1営業日以内、最短3時間でサービスのご利用が可能です。まずはお気軽にお問い合わせください。
